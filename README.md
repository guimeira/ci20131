Controle Inteligente 2013/1
=====================

English
---------

This is the source code of a simple project developed for the Intelligent Control subject at Federal University of Espirito Santo.

It's a pretty simple project using single-layer perceptrons to implement logic circuits.
In this project, I've built an 8-bit adder.

The `Perceptron` class can be useful for someone, as it implements a generic single-layer perceptron with a user-defined number of inputs.

The other classes are probably too simple or too specific to be useful for other projects.

Anyway, feel free to fork it :)

Português
-------------
Este é o código fonte de um projeto simples desenvolvido para a disciplina de Controle Inteligente na Universidade Federal do Espírito Santo.

É um projeto bem simples que utiliza perceptrons monocamada para implementar circuitos lógicos. Neste projeto, eu implementei um somador de 8 bits.

A classe `Perceptron` pode ser útil pra alguém, já que ela implementa um perceptron monocamada genérico com o número de entradas que o usuário desejar.

As outras classes provavelmente são muito simples ou muito específicas para serem úteis a outros projetos.

De qualquer forma, sinta-se à vontade para fazer um fork :)
